# Roadmap

## 0.1 — Public foundation

- Native macOS app, menu bar, quota telemetry, hybrid stop ladder, notifications, Keychain, CI, source packaging.

## 0.2 — Reliability

- Signed and notarized releases.
- On-device selector diagnostics with explicit user consent.
- Additional UI-localization coverage.
- Import/export of non-secret settings.

## 0.3 — Observability

- Optional local trend charts and reset calendar.
- Structured diagnostic bundle with automatic secret scrubbing.
- Configurable per-bucket policies.

## Later

- Safe updater after signing/notarization is established.
- Additional notification providers through isolated adapters.
- Broader automated UI compatibility matrix.

Roadmap items are intentions, not commitments. Safety and compatibility take priority over feature count.
